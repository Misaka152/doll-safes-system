# 萌趣玩偶销售系统开发文档

## 项目概述

萌趣玩偶销售系统是一个基于Vue 3开发的现代化电商平台，专注于玩偶产品的在线销售。系统提供了完整的用户注册登录、商品浏览、购物车管理、收藏夹等功能，并采用了模拟数据API进行前端开发和测试。

项目采用了现代化的UI设计，以粉色系为主色调，打造了温馨可爱的视觉体验，与玩偶产品的特性相契合。同时，系统注重用户体验，提供了响应式布局，确保在不同设备上都能获得良好的使用体验。

## 技术栈

- **前端框架**：Vue 3（使用Composition API）
- **状态管理**：Pinia
- **路由管理**：Vue Router
- **UI组件库**：Element Plus
- **构建工具**：Vite
- **CSS预处理**：原生CSS（采用作用域样式）
- **图标系统**：Element Plus Icons
- **模拟数据**：本地模拟API

## 系统架构

项目采用了典型的前端单页应用(SPA)架构，主要分为以下几部分：

1. **视图层(Views)**：包含各个页面组件
2. **组件层(Components)**：封装可复用的UI组件
3. **状态管理层(Stores)**：使用Pinia进行集中状态管理
4. **路由层(Router)**：处理页面导航和路由守卫
5. **数据层(Data)**：提供模拟API和数据

### 目录结构

```
src/
├── App.vue                # 应用根组件
├── assets/                # 静态资源
├── components/            # 可复用组件
│   ├── layout/           # 布局组件
│   └── product/          # 产品相关组件
├── data/                  # 模拟数据和API
├── router/                # 路由配置
├── stores/                # Pinia状态管理
└── views/                 # 页面视图组件
```

## 功能模块

### 1. 用户模块

#### 1.1 用户状态管理

用户状态管理通过`useUserStore`实现，主要功能包括：

- 用户登录认证
- 用户登出
- 用户信息维护
- 登录状态持久化（使用localStorage）

关键代码示例：
```js
// 登录功能
const login = (email, password) => {
  // 验证逻辑...
  
  // 更新用户信息
  user.value = userData
  token.value = 'mock-token-' + Date.now()
  
  // 本地存储
  localStorage.setItem('user', JSON.stringify(userData))
  localStorage.setItem('token', token.value)
  
  return { success: true, message: '登录成功' }
}
```

#### 1.2 路由守卫

系统实现了路由守卫，保护需要登录才能访问的页面：

```js
// 路由守卫，检查用户是否已登录
router.beforeEach((to, from, next) => {
  const userStore = useUserStore()
  
  if (to.meta.requiresAuth && !userStore.isLoggedIn) {
    next({ name: 'Login', query: { redirect: to.fullPath } })
  } else {
    next()
  }
})
```

### 2. 产品模块

#### 2.1 产品状态管理

产品状态通过`useProductStore`实现，主要功能包括：

- 获取产品列表
- 产品详情查看
- 产品分类筛选
- 产品搜索
- 精选产品和新品展示

关键计算属性：
```js
// 精选产品
const featuredProducts = computed(() => {
  return products.value.filter(p => p.isFeatured === true) || []
})

// 新品
const newProducts = computed(() => {
  return products.value.filter(p => p.isNew === true) || []
})
```

#### 2.2 产品展示组件

`ProductCard.vue`组件封装了产品展示卡片，实现了：

- 产品基本信息展示
- 折扣价格计算与显示
- 收藏功能
- 加入购物车功能
- 库存状态显示

### 3. 购物车模块

购物车功能通过`useCartStore`实现，主要功能包括：

- 添加商品到购物车
- 更新购物车商品数量
- 移除购物车商品
- 清空购物车
- 购物车总价和总数量计算

### 4. 收藏夹模块

收藏夹功能通过`useFavoritesStore`实现，允许用户收藏和管理喜爱的产品。

## 开发流程

### 1. 环境搭建

1. **安装Node.js**：确保安装Node.js 14.0以上版本
2. **克隆项目**：`git clone [项目仓库地址]`
3. **安装依赖**：`npm install`
4. **启动开发服务器**：`npm run dev`

### 2. 开发规范

#### 2.1 组件开发规范

- 使用Vue 3组合式API（Composition API）
- 组件文件采用PascalCase命名（如ProductCard.vue）
- 使用`<script setup>`语法
- 组件内样式使用`scoped`作用域隔离

#### 2.2 状态管理规范

- 使用Pinia进行状态管理
- 每个功能模块创建独立的Store
- 使用组合式API风格定义Store

#### 2.3 API调用规范

- 所有API请求封装在`data`目录下
- 使用统一的响应格式`{code, data, message}`
- 在Store中处理API调用和错误处理

### 3. 测试策略

- 组件单元测试
- 状态管理逻辑测试
- 端到端用户场景测试
